# Realtor AI Agent

An AI-powered real estate lead qualification and handoff platform built for portfolio and product demonstration purposes.

## Features
- Website chat interface
- Buyer/seller/renter/investor lead classification
- Structured lead capture
- AI-generated lead summaries
- Human handoff flow
- Booking prompt flow
- Firebase-backed lead storage
- Compliance-aware prompting

## Tech Stack
- Python
- FastAPI
- OpenAI API
- Firebase Firestore
- HTML / CSS / JavaScript
- GitHub
